🎯このアプリでできること

  • スマホがゲームパッドに変身：DCS Worldなどのゲームをスマホから操作
  • OBSを遠隔操作：録画開始/停止、シーン切替をスマホから
  • 自由にカスタマイズ：好きなボタンを好きな配置で作成可能

📋必要なもの

✅ 準備するもの

  1. Windows PC（Windows 10/11）
  2. スマホ（iPhone/Android どちらでもOK）
  3. 同じWi-Fi（PCとスマホが同じネットワークに接続）

📥 インストールが必要なもの

  • Python 3.10 ～ 3.12（無料）
  • このアプリ（ZIPファイル）
⚠️ Python バージョンについて
  • 推奨: Python 3.10、3.11、3.12
  • 非推奨: Python 3.14（最新すぎて一部ライブラリが未対応）

🚀セットアップ手順

ステップ1Pythonをインストール

方法A: 公式サイトからインストール（推奨）

  1. Pythonの公式サイトにアクセス
    • https://www.python.org/downloads/
  2. Python 3.11 または 3.12 をダウンロード
    • ページを下にスクロールして「Python 3.11.x」または「Python 3.12.x」を選択
    • 黄色いダウンロードボタンをクリック
  3. インストーラーを実行
    • ダウンロードした python-3.11.x.exe または python-3.12.x.exe をダブルクリック
  4. ⚠️ 重要：チェックボックスを確認
    • 画面下部の「Add Python to PATH」に必ずチェックを入れる
    • 「Install Now」をクリック
  5. インストール完了を待つ
    • 「Setup was successful」と表示されたら完了

方法B: Microsoft Storeからインストール

  1. Microsoft Storeを開く
    • Windowsキーを押して「store」と入力
  2. 「Python 3.11」または「Python 3.12」を検索してインストール
  3. ⚠️ 注意：コマンドが異なります

    この方法の場合、後述のコマンドで python の代わりに python3 または py を使用

ステップ2アプリを配置

  1. ZIPファイルを解凍
    • dcs-obs-controller.zip を右クリック
    • 「すべて展開」を選択
    • 好きな場所に解凍（例：デスクトップ）
  2. フォルダの中身を確認 
    dcs-obs-controller/
├── app.py              ← メインプログラム
├── config.json         ← 設定ファイル
├── requirements.txt    ← 必要なライブラリ一覧
└── templates/          ← 画面デザイン

ステップ3必要なライブラリをインストール

  1. コマンドプロンプトを開く
    • キーボードの Windowsキー を押す
    • 「cmd」と入力
    • 「コマンドプロンプト」をクリック
  2. フォルダに移動

    以下のコマンドを入力（解凍した場所に合わせて変更）

    cd C:\Users\あなたの名前\Desktop\dcs-obs-controller
    💡 簡単な方法：
    • エクスプローラーでフォルダを開く
    • アドレスバーに「cmd」と入力してEnter
  3. ライブラリをインストール
    ⚠️ 重要：正しいコマンドを使用

    ❌ py install -r requirements.txt ← 間違い

    ✅ pip install -r requirements.txt ← 正しい

    公式サイト版の場合：

    pip install -r requirements.txt

    Microsoft Store版の場合：

    pip3 install -r requirements.txt

    または

    py -m pip install -r requirements.txt

    または

    python -m pip install -r requirements.txt

    自動でインストールが始まります（30秒〜1分）

ステップ4アプリを起動

  1. コマンドプロンプトで実行

    公式サイト版の場合：

    python app.py

    Microsoft Store版の場合：

    python3 app.py

    または

    py app.py
  2. 起動成功の確認

    以下のような表示が出ればOK：

    ==================================================
DCS/OBS Controller Server
==================================================
PC用設定画面: http://localhost:5000/pc
スマホ用画面: http://[PCのIP]:5000/
==================================================
[INFO] Using 'pynput' library
[INFO] Key controller initialized
 * Running on http://192.168.x.x:5000

    この 192.168.x.x の数字をメモ

  3. ⚠️ Windowsファイアウォールの警告が出たら

    「アクセスを許可する」をクリック

ステップ5スマホから接続

  1. スマホのブラウザを開く
    • Safari（iPhone）またはChrome（Android）
  2. アドレスバーに入力 
    http://192.168.x.x:5000

    192.168.x.x はステップ4でメモした数字

  3. 画面が表示されればOK！
    • ボタンが並んだ画面が表示されます

⚙️初期設定

PCで設定画面を開く

  1. ブラウザで以下にアクセス 
    http://localhost:5000/pc
  2. 設定画面が開きます

🎮使い方マニュアル

📝 ボタンの追加方法

  1. PC設定画面で「➕ ボタンを追加」をクリック
  2. 情報を入力
    項目 説明 例
    ボタン名 スマホに表示される名前 「ギア」「録画開始」
    キー 押すキー g、F13、ctrl+s
    色 ボタンの色（青/緑/橙/赤） 青
  3. 「確定」をクリック
  4. 「💾 設定を保存」をクリック

🎨 タブ（モード）の使い方

タブとは？

  • ゲームごとに違うボタン配置を保存できる機能
  • 例：「DCS用」「OBS用」「配信用」

タブの追加

  1. 「➕ タブを追加」をクリック
  2. タブ名を入力（例：「OBS制御」）
  3. ボタンを追加

タブの切り替え

  • PC画面：上部のタブをクリック
  • スマホ画面：上部のタブをタップ

タブの編集・削除

  • タブ名の横の ✏️ で名前変更
  • タブ名の横の 🗑️ で削除

🎯OBSと連携する方法

1. OBSのホットキー設定

  1. OBSを起動
  2. 「設定」→「ホットキー」を開く
  3. 各機能にキーを割り当て
    機能 おすすめキー
    録画開始/停止 F13
    配信開始/停止 F14
    シーン1 F15
    シーン2 F16
    マイクミュート F17
  4. 「適用」→「OK」をクリック

2. アプリにボタンを追加

  1. PC設定画面を開く
  2. 「➕ タブを追加」で「OBS制御」タブを作成
  3. ボタンを追加

    例：

    • ボタン名：録画開始/停止
    • キー：F13
    • 色：赤

3. スマホから操作

  • OBSがバックグラウンドでも動作します
  • ゲーム中でもスマホから録画開始/停止が可能

🔧キーの指定方法

基本のキー

a, b, c ... z        → アルファベット
0, 1, 2 ... 9        → 数字
space                → スペース
enter                → エンター
esc                  → エスケープ

ファンクションキー

F1, F2 ... F24       → F1〜F24キー

組み合わせキー（pynputのみ）

ctrl+c               → Ctrl + C
shift+a              → Shift + A
alt+tab              → Alt + Tab
ctrl+shift+s         → Ctrl + Shift + S

❓トラブルシューティング

🔴 コマンドエラー

エラー: py install -r requirements.txt が動かない

原因：コマンドが間違っています
正しいコマンド： 
pip install -r requirements.txt

または

python -m pip install -r requirements.txt

または

py -m pip install -r requirements.txt

🔴 Python 3.14 でエラーが出る

原因：Python 3.14 は最新すぎて一部のライブラリが未対応
解決方法：
  1. Python 3.11 または 3.12 をインストール
    • https://www.python.org/downloads/
    • ページ下部から「Python 3.11.x」または「Python 3.12.x」を選択
  2. または、pynput のみを使う 
    pip install Flask flask-socketio flask-cors pynput

🔴 ModuleNotFoundError: No module named 'flask_socketio'

原因：ライブラリがインストールされていない
解決方法： 
pip install flask-socketio

または一括インストール：

pip install Flask flask-socketio flask-cors pynput

スマホから接続できない

原因1: PCとスマホが違うWi-Fiに接続している

解決：同じWi-Fiに接続し直す

原因2: ファイアウォールがブロックしている

解決：
  1. Windowsの「設定」→「プライバシーとセキュリティ」→「Windowsセキュリティ」
  2. 「ファイアウォールとネットワーク保護」
  3. 「ファイアウォールによるアプリケーションの許可」
  4. 「Python」を探して、プライベートとパブリックの両方にチェック

原因3: IPアドレスが間違っている

解決：
  • コマンドプロンプトで ipconfig を実行
  • 「IPv4アドレス」の数字を確認

ボタンを押しても反応しない

原因1: キーの指定が間違っている

解決：大文字・小文字、スペースを確認

原因2: ゲームがキーを認識していない

解決：ゲーム内のキー設定を確認

原因3: 管理者権限が必要（Windows）

解決：PowerShellを管理者として実行してから起動 
python app.py

アプリが起動しない

エラー1: python は、内部コマンドまたは外部コマンド...

原因：Pythonがインストールされていない、またはPATHが通っていない
解決方法：
  1. まず、どのコマンドが使えるか確認 
    python --version

    ダメなら

    python3 --version

    ダメなら

    py --version
  2. どれかでバージョンが表示されればOK
    • 例：Python 3.11.0
    • そのコマンドを使ってアプリを起動
  3. すべてダメな場合
    • Pythonを再インストール
    • 公式サイト版で「Add Python to PATH」に必ずチェック

💡便利な使い方

自動起動の設定

  1. バッチファイルを作成
    • メモ帳を開く
    • 以下を入力：

    公式サイト版の場合：

    @echo off
cd C:\Users\あなたの名前\Desktop\dcs-obs-controller
python app.py
pause

    Microsoft Store版の場合：

    @echo off
cd C:\Users\あなたの名前\Desktop\dcs-obs-controller
py app.py
pause
    • 「start_controller.bat」として保存
  2. ダブルクリックで起動

スマホのホーム画面に追加

iPhone:

  1. Safariでアプリを開く
  2. 共有ボタン→「ホーム画面に追加」

Android:

  1. Chromeでアプリを開く
  2. メニュー→「ホーム画面に追加」

📞よくある質問

Q: 複数のスマホから同時に接続できますか？

A: はい、できます。

Q: PCがスリープ状態でも動作しますか？

A: いいえ、PCが起動している必要があります。

Q: インターネット接続は必要ですか？

A: いいえ、同じWi-Fi内で動作します（オフラインOK）。

Q: DCS World以外でも使えますか？

A: はい、任意のキーボード操作が可能なので、あらゆるゲームやアプリで使用できます。
